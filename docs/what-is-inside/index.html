What's inside?

Libraries, frameworks and tools used in Traxion​

Traxion is a suite of orchestrated NodeJS packages. Here is the list of libraries, frameworks and tools used in Traxion:

Top libraries & frameworks​

Angular

A front-end framework for modern web.

Ant Design

A design system for enterprise-level products. Create an efficient and enjoyable work experience.

Docusaurus

A documentation framework built on top of React and MDX.

Hapify

Template engine for CRUD operations.

Jest

A JavaScript testing framework with a focus on simplicity.

NestJS

A progressive Node.js framework for building efficient, reliable and scalable server-side applications.

Ng Zorro

An enterprise-class Angular UI component library based on Ant Design, all components are open source and free to use under MIT license.

Nx

Next generation build system with first class monorepo support and powerful integrations.

React Admin

React-admin offers the best developer experience, lets you focus on business needs, and build delightful user interfaces.

Tailwind CSS

A utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.

Terraform

An open-source infrastructure as code software tool that provides a consistent CLI workflow to manage hundreds of cloud services.

TypeScript

A strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.

Important libraries​

Apollo

Apollo is a platform for building a unified graph, a communication layer that helps you manage the flow of data between your application clients (such as web and native apps) and your back-end services.

Casl

Isomorphic Authorization JavaScript library.

CDKtf

Cloud Development Kit for Terraform allows you to define infrastructure using a familiar programming language such as TypeScript, Python, or Go.

Class transformer

Allows you to transform plain object to some instance of class and versa.

Class validator

Allows use of decorator and non-decorator based validation.

Commander

Solution for node.js command-line interfaces.

Cypress

Fast, easy and reliable testing for anything that runs in a browser.

Enquirer

Stylish CLI prompts that are user-friendly, intuitive and easy to create.

ESlint

Statically analyzes your code to quickly find problems. Many problems ESLint finds can be automatically fixed. It runs automatically when commiting changes, thanks to Husky and Lint-staged. If an error cannot be fixed automatically, the commit will fail.

Express

Fast, unopinionated, minimalist web framework for node.

GraphQL

GraphQL is a query language for APIs and a runtime for fulfilling those queries with your existing data.

Husky

A tool triggering actions on git hook (before/after git commit, etc.).

JWT

JSON Web Tokens are an open, industry standard RFC 7519 method for representing claims securely between two parties.

Lodash

A modern JavaScript utility library delivering modularity, performance & extras.

MDX

MDX allows you to use JSX in your markdown content.

Minio

The MinIO JavaScript Client SDK provides simple APIs to access any Amazon S3 compatible object storage server.

Morgan

HTTP request logger middleware for node.js

Passport

Simple, unobtrusive authentication for Node.js.

Pg

Non-blocking PostgreSQL client for Node.js. Pure JavaScript and optional native libpq bindings.

Prettier

A unified code formatter that supports many languages, integrated in many IDE and has few options. It runs automatically when commiting changes, thanks to Husky and Lint-staged.

Prisma

Node.js and TypeScript ORM.

RxJS

RxJS is a library for composing asynchronous and event-based programs by using observable sequences.

Schematics

A scaffolding library for the modern web.

Supertest

HTTP assertions made easy via superagent.

Swagger

The Single Source of Truth for API Development Accelerating API delivery and quality through standards and collaboration, built on OpenAPI.

TS Essentials

A set of high-quality, useful TypeScript types that make writing type-safe code easier.

Webpack

A static module bundler for modern JavaScript applications.

Zx

Provides useful wrappers around child_process, escapes arguments and gives sensible defaults.

More stuffs​

@jscutlery/semver

Nx plugin for versioning using [SemVer](https://semver.org/) and CHANGELOG generation powered by [Conventional Commits](https://conventionalcommits.org).

Autoprefixer

PostCSS plugin to parse CSS and add vendor prefixes to CSS rules using values from Can I Use.

Case

An extensible utility to convert, identify, and flip string case.

Commitizen

A tool to define a standard way of committing rules and communicating it.

Commitlint

Forces the commit message format. It is required for automatic changelog generation and automatic versioning based on commit messages.

Constructs

Constructs are classes which define a "piece of system state". Constructs can be composed together to form higher-level building blocks which represent more complex state.

Cz conventional changelog

Part of the commitizen family. Prompts for conventional changelog standard.

Deepmerge

Merges the enumerable properties of two or more objects deeply.

Dotenv

A zero-dependency module that loads environment variables from a .envfile into process.env.

Faker

Generate massive amounts of fake data in the browser and Node.js.

FS extra

Adds file system methods that aren't included in the native fs module and adds promise support to the fs methods.

Glob

Match files using the patterns the shell uses, like stars and stuff.

Is-ci

A simple tool that detects if the current environment is a Continuous Integration server.

Lint-staged

A simple library that runs linters (eslint, prettier) on staged files before commiting.

Mime Types

A JavaScript content-type utility.

Nanoid

A tiny, secure, URL-friendly, unique string ID generator for JavaScript.

Ng-packagr

Compile and package Angular libraries in Angular Package Format (APF).

Ngx Translate

The internationalization (i18n) library for Angular.

Postcss

A tool for transforming styles with JS plugins.

SWC

A super-fast TypeScript / JavaScript compiler written in Rust.

Tslib

A runtime library for TypeScript that contains all of the TypeScript helper functions.

Uuid

For the creation of RFC4122 UUIDs.

Xhr2

An npm package that implements the W3C XMLHttpRequest specification on top of the node.js APIs.

Zone.js

Implements Zones for JavaScript, inspired by Dart.
